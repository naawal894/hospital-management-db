# 🏥 Hospital Management System (HMS)

A robust desktop application developed in **C#** using **Windows Forms** and integrated with a **SQL Database**. This system is designed to streamline hospital administration by managing patient records, doctor information, and appointment scheduling.

## ✨ Features
* **Patient Registration:** Add, update, and manage detailed patient profiles.
* **Doctor Management:** Track doctor specialties and availability.
* **Appointment Scheduling:** Digital logging of patient-doctor consultations.
* **Database Integration:** Reliable data storage using relational database management.
* **Search Functionality:** Quickly find records using specialized search queries.

## 🛠️ Tech Stack
* **Language:** C# (.NET Framework)
* **UI:** Windows Forms (WinForms)
* **Database:** SQL Server / MySQL
* **Tools:** Visual Studio

## 🚀 How to Run
1. **Clone the repo:** Download this repository to your local machine.
2. **Open the Solution:** Double-click `project_DB_FORMS.sln` to open it in Visual Studio.
3. **Setup Database:** - Run the SQL scripts provided in the `/database` folder in your SQL management tool.

4. **Build & Run:** Press `F5` or click **Start** in Visual Studio.
